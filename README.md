# Project 3: Email Marketing Campaign Analysis

## Overview
This project analyzes email marketing campaign data to evaluate performance and user engagement. The analysis is presented using an interactive Power BI dashboard to help drive data-informed marketing decisions.

---

## Objectives
* **Measure Performance:** Evaluate the overall success of email campaign initiatives.
* **Identify Trends:** Distinguish between top and low-performing campaigns to refine strategy.
* **Engagement Insights:** Understand user interaction patterns over time.

---

## Tools Used
* **Power BI:** Data visualization and dashboard development.
* **Excel / CSV:** Data source management and preprocessing.

---

## Key Metrics

| Metric | Description |
| :--- | :--- |
| **Emails Sent** | Total number of emails successfully delivered. |
| **Open Rate** | The percentage of recipients who opened the email. |
| **Click-Through Rate (CTR)** | The percentage of recipients who clicked a link within the email. |
| **Conversion Rate** | The percentage of recipients who completed the target action. |
